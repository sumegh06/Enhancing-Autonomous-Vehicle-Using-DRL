# Enhancing Autonomous Vehicle using Deep Reinforcement Learning

## Overview
This project implements an autonomous driving system using Deep Reinforcement Learning (DRL), where an agent learns to navigate and avoid obstacles through reward-based training.

## Key Features
- Lane navigation
- Obstacle avoidance
- Real-time decision making
- Reward-based learning system

## Tech Stack
- Python
- TensorFlow
- NumPy
- Pandas

## Methodology
The system uses reinforcement learning where:
- The agent interacts with the environment
- Receives rewards based on actions
- Learns optimal driving behavior over time

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the main file:
   python main.py

## Results
- Improved navigation performance in simulated environment
- Better decision-making through continuous learning
